Open Bidder

Open Bidder is an open source real-time bidding (RTB) framework that enables developers to quickly build and deploy a robust real-time bidder.

Learn more

1. Sign Up

Create a Google API Project with billing enabled and sign up for:

  • Google Compute Engine
  • Google Cloud Storage
  • Ad Exchange Buyer API
  • Google BigQuery

Your DoubleClick Ad Exchange Account Manager can guide you through this process.

2. Create a Project

Create a new project. You will need:

  • Google API Project ID
  • Google API Project Number
  • OAuth 2.0 client secret and ID

Additional configuration may be required for your own bidding interceptors and if you wish to receive impression and click callbacks.

3. Start Bidding

Open Bidder will start load balancers and bidders on behalf of you and your team. You can extend the bidder with your own custom bidding logic and also reuse bundled functionality. Once registered with the exchange your bidders will receive bid requests for your pre-targeted campaigns.
