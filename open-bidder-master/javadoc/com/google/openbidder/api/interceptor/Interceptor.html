  • Type Parameters:

    All Known Implementing Classes:

    public interface Interceptor<Req extends UserRequest,Resp extends UserResponse<Resp>>
    Root interceptor interface.

    Interceptors may inject various Open Bidder provided dependencies using Guice.

    Interceptors may annotate methods with PostConstruct if they wish to take some actions prior to handling requests and PreDestroy if they wish to take actions after the interceptor is handling requests and before the controller shuts down.
