        B setMediaType(com.google.common.net.MediaType mediaType)
        Sets the message's content type (Content-Type header).

        This includes the character encoding; if absent, UTF-8 will be used as default encoding.

      • printContent

        B printContent(String s)
        Adds text payload as a String; has same effect as contentWriter().print() (will use the same PrintWriter object, subject to the same buffering). This method is convenient because it enables fluent call chains (good for tests), but PrintWriter provides a much richer API for writing text data.

        You can use this method interchangeably with all other content-setting methods.
