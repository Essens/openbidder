Package com.google.openbidder.deals

Preferred Deals support for DoubleClick Ad Exchange.

See: Description

  • Interface Summary 
    Interface Description
    PreferredDealCollectionManager
    Manages the PreferredDealCollection, keeping it up-to-date with some external data source.
    PreferredDealUpdateListener
    Implemented by classes that wish to be notified of updates to the Deals.PreferredDeals.
  • Class Summary 
    Class Description
    AbstractPreferredDealCollectionManager
    Exchange- and data source-neutral implementation for PreferredDealCollectionManager.
    CloudStoragePreferredDealCollectionManager
    Populates the PreferredDealCollection with preferred deals information from the Storage Service.
    NullPreferredDealCollectionManager
    No-op PreferredDealCollectionManager.
    PreferredDealCollection
    An in-memory collection of Deals.PreferredDeals.
    PreferredDealsInterceptor
    An interceptor that bids on Ad Exchange Preferred Deals.
    PreferredDealsModule
    Provides the PreferredDealCollectionManager and PreferredDealCollection.
    PreferredDealUpdateEvent
    Event sent when Deals.PreferredDeals have been updated from the storage service (the two collections should be mutually exclusive)..
  • Annotation Types Summary 
    Annotation Type Description
    PreferredDealsBucket
    Preferred deals bucket name.
